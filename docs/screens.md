# STREAK 画面一覧

## 1. 公開画面

| 画面 | URL 案 | 利用者 | 目的 |
| --- | --- | --- | --- |
| LP | `/` | 加盟店候補 | STREAK の概要、対応決済、導入導線を表示する |
| 加盟店登録 | `/signup` | 加盟店 | メールアドレス / パスワードで登録する |
| ログイン | `/login` | 加盟店 / 管理者 | ダッシュボードへログインする |
| メール認証案内 | `/auth/verify-email` | 加盟店 | メール確認待ち状態を案内する |
| 決済リンク | `/pay/[linkId]` | 購入者 | 商品確認、購入者情報入力、USDC 決済を行う |
| 決済完了 | `/pay/[linkId]/complete` | 購入者 | 決済完了と TX ハッシュを表示する |
| 決済失敗 / 中断 | `/pay/[linkId]/failed` | 購入者 | 決済失敗または中断時の案内を表示する |

## 2. 加盟店画面

| 画面 | URL 案 | 主な機能 |
| --- | --- | --- |
| ダッシュボード | `/dashboard` | 売上合計、注文数、最近の決済を表示 |
| 初期設定 | `/dashboard/onboarding` | 店舗情報、ウォレット、規約同意状態の確認 |
| 商品一覧 | `/dashboard/products` | 商品一覧、公開状態、決済リンク導線 |
| 商品作成 | `/dashboard/products/new` | 商品名、説明、価格 USDC を登録 |
| 商品編集 | `/dashboard/products/[productId]` | 商品情報の編集、停止 |
| 決済リンク一覧 | `/dashboard/payment-links` | 発行済みリンク、コピー、状態確認 |
| 注文履歴 | `/dashboard/orders` | 購入者情報、商品、金額、注文状態を確認 |
| 決済履歴 | `/dashboard/payments` | TX ハッシュ、配分額、決済状態を確認 |
| 設定 | `/dashboard/settings` | 店舗情報、担当者情報、受取ウォレットを更新 |

## 3. 管理者画面

| 画面 | URL 案 | 主な機能 |
| --- | --- | --- |
| 管理ダッシュボード | `/admin` | 総売上、手数料、加盟店数、直近決済を表示 |
| 加盟店一覧 | `/admin/merchants` | 加盟店検索、有効 / 停止、手数料率確認 |
| 加盟店詳細 | `/admin/merchants/[merchantId]` | 店舗情報、ウォレット、商品、注文、手数料率を確認 |
| 注文一覧 | `/admin/orders` | 全注文の検索、状態確認 |
| 決済履歴 | `/admin/payments` | 全決済の TX、金額、配分を確認 |
| 手数料集計 | `/admin/fees` | 期間別・加盟店別の STREAK 手数料集計 |
| 監査ログ | `/admin/audit-logs` | 管理操作の履歴確認 |

## 4. 購入者決済画面の流れ

1. 決済リンクを開く。
2. 商品名、加盟店名、価格 USDC を確認する。
3. 購入者情報を入力する。
4. ウォレットを接続する。
5. USDC 支払いを承認する。
6. 決済完了画面で TX ハッシュを確認する。
7. 決済完了メールを受け取る。

## 5. 加盟店オンボーディングの流れ

1. LP から加盟店登録へ進む。
2. メールアドレス、パスワードを入力する。
3. メール認証を完了する。
4. 店舗情報を入力する。
5. 受取ウォレットを設定する。
6. 規約同意と反社確認を行う。
7. 商品を登録する。
8. 決済リンクをコピーして販売導線に設置する。
